# Esharq contributors

`Esharq.png` — the shared Esharq Contributor badge (256×256 circular EA monogram). Everyone
listed in `EsharqContributors` (in the client's `src/utils/constants.ts`, seeded with the whole
dev team) gets this image in their profile, rendered with a spinning RGB rainbow ring — the same
animated design as the Custom badges.

The recipient list is code-driven (the `EsharqContributors` set), so unlike the Custom badges
there is no JSON map here. To grant the badge to someone, add their Discord id to that set in the
client. The image is the same for every contributor.
